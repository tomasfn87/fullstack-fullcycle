# fullstack-fullcycle

Objetivos:
----------
* Armazenar arquivos do curso FullStack da FullCycle
* O curso se trata de uma imersão para criação de uma aplicação financeira (compra, pagamento, etc)
* Assuntos abordados:
  - Container ( Docker )
  - Go ( Go Lang )
  - JavaScript ( Nest.js, Next.js (React) )
  - Apache/Kafka
  - Clean Architecture
